# エコクリーン ランディングページ

エアダクトクリーニングサービス「エコクリーン」のランディングページです。

## ファイル構成

```
├── index.html    ... メインLP（HTMLマークアップ）
├── css/
│   └── style.css ... スタイルシート（レスポンシブ対応含む）
├── images/       ... 画像ディレクトリ（プレースホルダー）
└── README.md     ... このファイル
```

## ブラウザでの確認方法

`index.html` をブラウザで直接開くとプレビューできます。
Chrome DevToolsのデバイスモードでタブレット・スマホ表示も確認可能です。

## WordPress（SWELL）への設置手順

### 方法1: SWELL LP投稿タイプ（推奨）

1. WordPress管理画面にログイン
2. 左メニュー「LP」→「新規追加」
3. タイトルを入力（例：「エコクリーン ランディングページ」）
4. 右側パネルの「LP設定」で以下を設定：
   - ヘッダー：非表示
   - フッター：非表示（必要に応じて）
5. 「+」ボタン → 「カスタムHTML」ブロックを追加
6. **index.html の `<body>` 内のHTMLコードを貼り付け**
7. 別のカスタムHTMLブロックを追加し、以下を貼り付け：
   ```html
   <style>
   /* css/style.css の中身をここに貼り付け */
   </style>
   ```
8. 「公開」をクリック

### 方法2: 固定ページとして作成

1. 「固定ページ」→「新規追加」
2. テンプレートを「LP」に変更（SWELL対応テンプレート）
3. 以降は方法1の手順5〜8と同じ

## 画像の差し替え手順

1. WordPress管理画面「メディア」→「新規追加」で画像をアップロード
2. アップロード後、画像のURLをコピー
3. HTML内の `src="images/xxxxx.jpg"` の部分をコピーしたURLに書き換え

## カスタマイズ

### 色の変更
`css/style.css` 冒頭の `:root` セクションでCSS変数を変更できます。

### 電話番号・LINE IDの変更
`index.html` 内の以下の箇所を検索・置換してください：
- `0120-XXX-XXX` → 実際の電話番号
- `https://line.me/R/ti/p/XXXXX` → 実際のLINE友だち追加URL

## レスポンシブ対応

| デバイス | 幅 |
|---------|-----|
| PC | 1200px以上 |
| タブレット | 768px〜1199px |
| スマホ | 〜767px |
| 小型スマホ | 〜480px |

スマホ表示時は画面下部に固定CTAバー（電話・LINE）が表示されます。
